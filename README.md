# PdfBatchPrint

Windows環境で複数のPDFファイルを一括印刷するためのライブラリです。
PDFium (Docnet.Core) をエンジンとして採用しており、レイアウトを崩さず高品質な印刷が可能です。

## 🚀 主な機能

- **高品質印刷**: PDFを正確にラスタライズし、文字化けやレイアウト崩れを防ぎます。
- **自動レイアウト**: 用紙サイズへの自動フィット、およびPDFの縦横に応じた自動回転に対応。
- **柔軟な設定**: 部数、両面印刷（長辺/短辺）、用紙サイズの指定が可能です。
- **非同期・中断サポート**: `CancellationToken` による印刷処理の中断に対応しています。

## 📦 使い方の流れ

### 1. 印刷したいファイルと設定を準備する
印刷したいPDFファイルごとに、以下の項目をセットにしたリストを作成します。

- **ファイルの参照**: 印刷するPDFのフルパスを指定します。
- **ドキュメント名**: 印刷ジョブ（プリンターの待ち行列）に表示される名前です。
- **印刷枚数**: 必要な部数を指定します（デフォルトは1部）。
- **用紙サイズ**: A4、B5、ハガキなど、プリンターが対応しているサイズを選べます。
- **両面印刷**: 片面、長辺綴じ（本のようにめくる）、短辺綴じ（カレンダーのようにめくる）から選べます。

### 2. プリンターを選んで実行する
PCにインストールされているプリンター名（例：「Microsoft Print to PDF」や「複合機_1F」など）を指定して印刷を開始します。

---

## 🛠️ 各種設定（PdfPrintItem）の詳細

| 設定項目 | 日本語での意味 | 備考 |
| :--- | :--- | :--- |
| `FilePath` | **ファイルパス** | 印刷するPDFの場所（必須） |
| `FileName` | **表示名** | 印刷中に表示される名前 |
| `Copies` | **印刷枚数** | 1以上の数値を指定 |
| `PaperKind` | **用紙サイズ** | A4, B5, Letter など |
| `Duplex` | **両面/片面** | 片面, 長辺綴じ, 短辺綴じ |

## 🔍 便利な機能

- **プリンター一覧の取得**: PCに繋がっているプリンターの名前を一覧で取得できます。
- **用紙オプションの確認**: 選んだプリンターで「どのサイズが使えるか」「両面印刷ができるか」などを事前にチェックできます。

## 注意事項

- **OS 依存**: 本ライブラリは `System.Drawing.Printing` および Windows スプーラーを使用しているため、Windows OS 上でのみ動作します。
- **物理マージン**: プリンターのハードウェアが持つ物理的な余白（PrintableArea）を考慮して描画されます。
- **レンダリングスケール**: 高品質な印刷結果を得るため、内部的に高解像度でのラスタライズを行っています（デフォルトで DPI に基づくスケールまたは 2.0 倍以上）。
